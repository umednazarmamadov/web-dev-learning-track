# Web App Development Learning Track

A structured learning track focused on backend web development fundamentals.

## Topics Covered

- PHP Basics — variables, functions, OOP
- SQL — table design, queries, JOINs
- Git — version control workflow

## Goals

- Build strong backend fundamentals in PHP
- Practice database design and SQL queries
- Develop consistent Git workflow habits
